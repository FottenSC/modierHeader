# No Adware Policy

modierHeaders must remain a utility, not a promotion channel.

The project prohibits:

- install, update, or uninstall ad pages
- unsolicited tabs or windows
- partner redirects
- coupon, shopping, or commission-link injection
- hidden telemetry
- remote configuration that changes browser behavior
- remote executable scripts
- obfuscated-only release source

Any future monetization must be direct and user-initiated, such as paid local features, sponsorship, or donations. It must not alter browsing, links, cookies, headers, or tabs for advertising.
